数据分析

numpy

Numpy是python最基本最常用的科学计算库，在数据分析，机器学习，深度学习等领域经常被使用。numpy的主要对象是同质多维数组。它是一张表，所有元素（通常是数字）的类型都相同，并通过正整数元组索引。在numpy中，维度成为轴。轴的数目是rank。

例如[1,2,3]是rank为1 的数组，因为它具有一个轴，该轴的长度为3,。

上面这个实例中，rank为2（它是二维的）。第一维度（轴）的长度为2，第二维度的长度为3。

NumPy的数组的类成为ndarray，别名array。ndarray的属性如下：

  • ndarray.ndim：数组的轴（维度）的个数，在Python中，维度的数量被成为rank。
  • ndarray.shape：数组的维度，对于具有n行m列的矩阵，shape将是（n, m）。
  • ndarray.size：数组元素的总数，等于shape元素的乘积。
  • ndarray.dtype：描述数组中年元素类型的对象。比如，numpy.int32，numpy.int16和numpy.float64。
  • ndarray.itemsize：数组中每个元素的字节大小。例如元素为float64类型的数组的itemsize为8（64/8），而complex32类型的数组的comitemsize为4（32/8）

数组创建

常见的错误，用多个数字参数调用array，而不是提供单个数字列表作为参数。

  • >>> a = array(1,2,3,4) # wrong 。
  • >>> a = array([1,2,3,4]) # right 。

array将序列的序列转换成二维阵列，将序列的序列的序列转换成三维数组。等等…

数组的类型可以在创建时明确指定：

通常，数组的元素最初是未知的，但其大小是已知的。因此NumPy提供了几个函数来创建具有初始占位符内容的数组，这些最小化了数组增长的必要，数组增长是一种昂贵的操作。

函数zeros创建一个由0组成的数组，函数ones创建一个由1组成的数组，函数empty内容是随机的并且取决于存储器的状态，默认情况下，创建的数组的类型是float64。

为了创建数字序列，NumPy提供类似于range的函数，返回数组而不是列表。

当arange与浮点参数一起使用时，由于浮点数的精度是有限的，通常不可能预测获得的元素数量，出于这个原因，通常最好使用函数linspace，它接受我们想要的元素数量而不是步长作为参数。

打印数组

打印数组时，NumPy以类似嵌套列表的方式显示它，但是使用以下布局：

  • 最后一个轴从左到右打印；
  • 第二个到最后一个从上到下打印；
  • 其余的也从上到下打印，每一个切片与下一个用空行分开。

如果数组太大而无法打印，NumPy会跳过数组的中心部分，并只打印边角：

如果要强制打印整个数组，可以使用set_printoptions更改打印选项。

>>> np.set_printoptions(threshold='nan')

基本操作

数组上的算术运算使用元素级别。将创建一个新数组并 用结果填充。

与许多矩阵语言不同，乘法运算符*的运算在numpy数组中是元素级别的乘积，可以使用dot函数或方法执行矩阵乘积。

某些操作（+=或者*=）可以修改现有数组，而不是创建新数组。

当使用不同类型的数组操作时，结果数组的类型是更一般或更精确的数组（成为向上转换的行为）。

exp方法是以e为底的指数函数，括号里面是幂

np.exp(1)=e**1=2.71...

np.exp(3)=e**3

许多一元操作，例如计算数组中所有元素的和，被实现为ndarray 类的方法。

默认情况下，这些操作适用于数组，就像他是一个数字列表，而不管其形状。但是，通过指定axis参数，你可以沿着数组的指定轴进行操作。

axis参数是对数组shape的下标（也可以说是索引）进行操作，axis=0代表对shape下标也就是索引为0的进行操作，shape索引为0的数字是3，也就是行。那axis=0就是对行操作，上面的b.sum(axis=0)意思也就是所有行的和。同理axis=1就是对列进行操作。

NumPy提供了熟悉的数学函数，如sin，cos，exp等等。在NumPy中这些被称为“通用函数”（ufunc）。在NumPy中，这些函数在数组上按元素级别操作，产生一个数组作为输出。
