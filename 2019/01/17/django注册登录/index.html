django注册登录

发表于 2019-01-17 | 分类于 django

前言

用django框架做一个注册登录的表单，一个比较小的项目。

前期工作

  1. 创建项目

    • cd到项目文件夹，在项目文件夹下输入命令django-admin startproject login 来创建一个名为login的项目。
    • cd到我们创建的项目下输入python manage.py startapp myLogin 。

    这个时候我们的目录应该如下：

    • 在命令行输入python manage.py migrate迁移数据库

  2. settings更改

    • LANGUAGE_CODE = ‘zh-hans’ 默认en-us

    • TIME_ZONE = ‘Asia/Shanghai’ 默认UTC

    • 数据库配置如下：mysql -u root -p验证mysql是否能启动。

开发

  2. 在settings.py的INSTALLED_APPS中引入：’login.apps.LoginConfig’

  3. 迁移数据库 python manage.py migrate

    执行完这一步之后发现报错，这是因为缺少数据库的插件。

  4. 然后命令行，python manage.py makemigrations（这一步不能省略）之后在python manage.py migrate发现数据库多了一张表，就是我们创建的表。

  5. 创建管理员用户：python manage.py createsuperuser

  6. 运行项目，pthon manage.py runserver

  7. 访问domain: port/admin ，用上一步的后台账户登录.

    进去之后发现没有我们创建的表。

    所以我们需要在admin.py中把我们创建的表引入过去。

    然后刷新一下我们的管理员后台页面。发现多了一个用户表，后面多个s是因为django默认我们创建的表是复数。
