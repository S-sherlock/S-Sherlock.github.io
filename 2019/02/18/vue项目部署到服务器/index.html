步骤

  1. 克隆项目

  2. 进入项目目录执行npm i安装依赖

  3. 执行npm run dev跑项目

  4. 项目文件夹下npm run pub

  5. 然后在pycharm里配置服务器(tools/start ssh session)

  6. 打开项目发现新生成一个.public文件夹

  7. 把.public文件夹拖到云服务器中home/wangzheng/下

  8. 在etc/下找到nginx文件夹，修改nginx.conf文件。

  10. 配置好之后点upload。

  11. 然后开启服务就完了。
