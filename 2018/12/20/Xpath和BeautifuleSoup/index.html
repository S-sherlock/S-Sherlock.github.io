Xpath和BeautifuleSoup

发表于 2018-12-20 | 分类于 爬虫

BeautifuleSoup

bs包把html按照节点的层级关系转换为树形文档，然后解析，简单易用

beautifulSoup是一个第三方库

它是从html文件或xml文件里提取数据的库

安装

pip install beautifulsoup4 ，beautifulsoup只能用于python2

语法

首先引入 ，然后写一个简单的html。

html.parser python内置，方便兼容性好。lxml基于c，效率较高。

重要且常用的语法：

  1. find_all()，以列表的形式返回，括号内参数是标签名。
  1. 根据标签属性查找

因为class是在python中是关键字，所以在class后面加个下划线。

标签名.string，取出标签里的文本内容，如果该标签还有子标签，得到的是None

bs.标签名.attrs 获取标签的所有属性

其他不常用

  • print(bs.prettify()) 格式化输出网页内容
  • del bs.a['id'] 删除标签里的属性，此行代码是把id属性删除
  • print(bs.body.contents) 可以将标签的节点以列表的形式输出

获取body的子节点，并且第一行输出的是一个可迭代的对象

Xpath

先转文档树再解析 需要lxml包，lxml 是第三方的html解析库，在对HTML进行解析的时候，使用xpath语法。xpath语法通过路径的形式在HTML中选择节点，使用pip3 install lxml安装。

首先引入，然后再写一个简单的网页。

语法

常用且重要

/表示从文档顶层开始匹配，/body/ul/li

//表示匹配标签，目标标签可能有多级父节点。

搜索a标签 ，返回列表，每一项是元素对象。

提取标签属性值

获取标签内容，//标签/text()

查找时根据属性限制标签 ，待查标签[@属性名=属性值]

获取ul里的最后一个li标签里第一个a标签里的文本。

其他不常用的

  • 包含条件:contains(@class, “class值”) 适合没有id，name属性有class属性的情print(html.xpath('//div[@class="one"]/p[contains(@class, "first")]/text()'))
  • 查询下一级节点，省略中间的层级print(html.xpath('//ul//a[@class="first_a"]/text()'))
  • 根据css语法查找，最后一个节点print(html.xpath('//ul/li[last()]'))

重点用Xpath
